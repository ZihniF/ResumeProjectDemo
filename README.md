
## 🌟 Resume Project

## 🚀 Proje Özeti
   **Bu proje, M&Y Yazılım Eğitim Akademi Danışmanlık bünyesinde yürütülen eğitim kapsamında, Murat Yücedağ'ın rehberliğinde geliştirilmiştir.**  
   
🔷 Modern portfolyo sitelerinin ötesine geçen bu sistemde kullanıcılar, tüm profesyonel içeriklerini tek bir yönetim paneli üzerinden dinamik olarak yönetebiliyor. Statik sayfalar yerine tamamen veri odaklı bir yapı sunuluyor.

### 🔷 Gerçek Zamanlı İçerik Yönetimi
🔸 Projeler, yetenekler, referanslar anlık güncellenir   
🔸 Kod değişmeden içerik yönetimi  
🔸 Tam anlamıyla kişisel CMS sistemi  

### 🔷 Yönetim Paneli & Dashboard
🔸 Mesaj trafiği ve istatistikler tek ekranda  
🔸 Gelen talepler merkezi olarak yönetilir  
🔸 Portfolyo kategori dağılımı analiz edilebilir  

## 🔷 Modüler UI Mimarisi
🔸 ViewComponent yapısı ile tekrar kullanılabilir bileşenler  
🔸 “Son mesajlar”, “yetenekler” gibi dinamik modüller  
🔸 Temiz ve sürdürülebilir frontend yapısı  

## 🔷 Gelişmiş Kullanıcı Deneyimi
🔸 Responsive (mobil, tablet, desktop uyumlu)  
🔸 Animasyonlar, video background ve modern UI efektleri  
🔸 Profesyonel ve akıcı kullanıcı deneyimi  

## 🔷 Veri Odaklı Mimari
🔸 Tüm içerikler veritabanı üzerinden yönetilir  
🔸 Entity Framework Core ile güçlü veri erişimi  
🔸 Migration desteği ile versiyon kontrolü  

## 🔷 Temiz ve Ölçeklenebilir Mimari
🔸 ASP.NET Core MVC katmanlı yapı  
🔸 Controller – View – Model ayrımı  
🔸 Gerçek projelere direkt uygulanabilir mimari  

### Kullanılan Teknolojiler
 
🔸 **ASP.NET Core**  
🔸 **C#**    
🔸 **Entity Framework Core**    
🔸**SQL Server**   
🔸 **Razor / ViewComponents**    
🔸 **Bootstrap & SCSS**
